Frosted Glass Demo

For this demo, I will show you how to create this frosted glass effect in CSS. This glass blur adds a cool overlay and allows you to have an interesting background without it clashing with your text.

Something I would like to note is that I have found two possible methods to use for this effect. One of them is much simpler and requires less CSS, but it is compatible with very few browsers. For this reason, I will only show you the first one.

Body

You will first want to create a new div tag and give it the class container. We will return to this class later.

Next, link your CSS stylesheet. In that stylesheet, we will now be adding content to the body element. First, apply a background image of your choice. You will next apply this to your background:

            
                body {
                    background-attachment: fixed;
                }

This is necessary because the children of the body will inherit the image and we want it to always be the same size.

You can now experiment with some sizing and other aspects of the background, but this is how my final body element looks in my CSS stylesheet:

               
                 body {
                     background-image: url(images/image.jpg);
                     background-attachment: fixed;
                     background-size: cover;
                     display: grid;
                     justify-content: center;
                     height: 100vh;
                     background-position: center;
                     background-repeat: no-repeat;
                 }

Container

Going back to our container class, this is what we will use to actually create the frosted glass look. You can start off with applying a height and width for your box with the glass blur. I used 25rem for height and 40rem for width.

Next, apply some color with a box shadow:

                
                .container {
                    box-shadow: 0 0 1rem 0 rgba(0, 0, 0, .2);
                }

There are some other necessary things to apply to this class just do contain the blur:

                
                    .container {
                        position: relative;
                        z-index: 1;
                        overflow: hidden;
                    }

I then messed around with some margin values and a border radius just to change the look of the box. These parts are not necessary, but I think they improve how it looks.

Lastly, I set the background to inherit, so my final container class looks like this:

                
                    .container {
                        width: 40rem;
                        height: 25rem;
                        box-shadow: 0 0 1rem 0 rgba(0, 0, 0, .2);
                        border-radius: 5px;
                        position: relative;
                        z-index: 1;
                        background: inherit;
                        overflow: hidden;
                        margin: 100px auto;
                    }

But you'll notice that this has not created the frosted glass effect at all, just created a box with a shadow. To create the blur, we will use a pseudo element of our container class.

First we will apply a new box shadow, as well as the following content:

                
                    .container:before {
                        content: "";
                        position: absolute;
                        background: inherit;
                        z-index: -1;
                        top: 0;
                        left: 0;
                        right: 0;
                        bottom: 0; 
                        box-shadow: inset 0 0 2000px rgba(255, 255, 255, .5);
                    }

Finally, the last thing we need to apply to actually give it the blur is this:

                
                    .container:before {
                        filter: blur(10px);
                    }

And just like before, you can mess around with the spacing. My final pseudo element looks like this:

                
                    .container:before {
                        content: "";
                        position: absolute;
                        background: inherit;
                        z-index: -1;
                        top: 0;
                        left: 0;
                        right: 0;
                        bottom: 0; 
                        box-shadow: inset 0 0 2000px rgba(255, 255, 255, .5);
                        filter: blur(10px);
                        margin: -20px;
                    }

And there you go, you have successfully created the frosted glass look. If you would like to add text/content inside like what I'm doing, just put everything inside the div with the container class. From there you can mess around with the alignment and other stylistic elements of your text with CSS like normal.
